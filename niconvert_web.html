介绍

niconvert 是一个弹幕视频网站上的评论转换成 ass 字幕，方便在桌面播放器可识别的 ass 格式字幕，并提供简单的转换选项。

这个工具是自由软件，此为网页版，方便伸手党，也提供命令行版，更多信息可参阅 github 的 niconvert 项目。

目前支持 acfun 或 bilibili ，在下面输入视频地址然后抓取，接着设置选项，点击「下载」按钮即可。
